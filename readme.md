# Arquitectura Semantica y Layouts Modernos

Este repositorio presenta un **manifiesto de diseño** construido con HTML5 semantico y CSS Grid.

## Estructura del proyecto
- `index.html`: documento principal con etiquetas `header`, `main`, `section`, `article`, `aside` y `footer`.
- `css/style.css`: estilos con grilla de 6 columnas, variables CSS y tipografia externa.

## Jerarquia visual elegida
- Nivel 1: titulo principal de gran escala para abrir la narrativa del lookbook.
- Nivel 2: tarjetas principales en color de alto contraste (coral y lima) para dirigir la lectura.
- Nivel 3: tarjetas de soporte y bloque lateral (`aside`) para contexto curatoral.
- Nivel 4: pie de pagina con informacion institucional.

La jerarquia se refuerza con superposiciones controladas, cambios de tamano, color y peso tipografico.

## Guia rapida para principiantes
1. Edita textos y estructura en `index.html`.
2. Edita colores, tipografias y layout en `css/style.css`.
3. Si agregas una nueva seccion HTML, crea su clase CSS para estilizarla.
4. Revisa en navegador escritorio y movil para validar responsive.
